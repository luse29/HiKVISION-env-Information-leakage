# HiKVISION-env-Information-leakage
HiKVISION综合安防管理平台env信息泄漏漏洞poc
# 简介
适用于HiKVISION综合安防管理平台env信息泄漏漏洞

使用fofa测绘报表收集资产到url.txt文件里面即可，支持单条url扫描以及批量扫描

使用python启动（演示中使用的是python3）
